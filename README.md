### 极简导航

一个专注极简的导航主页。
网站： https://nav.yhz610.com/

#### 特点

- 内置百度、谷歌、必应、秘迹搜索引擎 
- 内置和风天气插件
- 主页背景图片调用biying壁纸，页面刷新会切换图片
- 侧栏汇集有邮箱、社交、视频、购物、视觉设计、集成部署、技术学习、实用工具等优质的网址
- 侧栏导航地址类目偏前端设计、技术开发
- 响应式布局
